# zklogin2

A zklogin implementation project.

## Description

This project implements zklogin functionality for secure authentication.

## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm or yarn

### Installation

```bash
npm install
```

## Usage

```bash
npm start
```
